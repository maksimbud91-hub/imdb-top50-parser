# 🎬 Парсер топ-50 фильмов с IMDb

Скрипт на Python, который автоматически собирает информацию о 50 лучших фильмах с сайта IMDb и сохраняет её в файл Excel.

## 🚀 Что делает программа
- Открывает страницу IMDb Top 250
- Собирает для каждого фильма: номер в рейтинге, название, год выпуска, длительность, возрастной рейтинг, оценку
- Показывает результат в консоли
- Сохраняет все данные в файл `imdb_top50.xlsx` (полная перезапись при каждом запуске)

## 📦 Используемые библиотеки
- `selenium` — для работы с браузером
- `pandas` — для создания таблицы Excel
- `time`, `os` — служебные
